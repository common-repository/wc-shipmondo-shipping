Multi-Carrier Shipmondo shipping plugin for WooCommerce displays live shipping rates at cart / checkout pages.

== Description ==
[Shipmondo](https://shipmondo.com/) is the shipping solutions company located in Denmark that offers out of the box integration with a number of European and Worldwide shipping carriers.

Shipmondo Shipping PRO for WooCommerce plugin utilizes [Shipmondo API](https://app.shipmondo.com/api/public/v3/specification) and improves your customers shopping experience by displaying live shipping rates at the cart / checkout pages.

= Are you shipping from Europe? =
Shipmondo is the best shipping solution for European e-commerce stores. 
Our FREE and PRO plugins will take your integration to the next level and offer a lot of features which will help to steamline your business operations.

== Features ==
* Support of multiple carriers
* Live shipping rates at WooCommerce cart & checkout with ability to control at what page rates are requested
* Min and max rate cost requirement for shipping rate to be displayed during checkout
* Control max number of live shipping rates that will be displayed in the cart and checkout pages
* Fetch shipping rates for each product or combine them to get better shipping rates
* Accommodate your packaging and marketing material with flexible package weight adjustments
* Charge extra handling fee or implement currency conversion with flexible shipping rate adjustments
* Easily find issues by enabling debug mode
* Speed up your website by caching and re-using previously found rates

== PRO Features ==
* Automatically email shipping label to the sender of a package
* Use WooCoommerce store or Marketplace / Dokan vendor’s location as the originating address for accurate shipping rates
* Create batches and print manifests with [Shipping Manifests PRO](https://1teamsoftware.com/product/woocommerce-shipping-manifests-pro/)
* WCFM vendors to can print shipping labels with [Shipping Labels for WCFM](https://1teamsoftware.com/product/woocommerce-shipping-labels-for-wcfm-pro/) add-on
* Dokan vendors to can print shipping labels with [Shipping Labels for Dokan](https://1teamsoftware.com/product/woocommerce-shipping-labels-for-dokan-pro/) add-on
* Dimension and weight based packing of the items into configurable boxes
* Ability to combine parcels to get better shipping rates
* Display different name for selected shipping services
* Allow all or only selected shipping services
* Define product conditions that have to be matched for service to be offered during checkout
* Configurable purchase postage workflow
* Buy shipping labels in just one click
* Bulk purchase and download of consolidated shipping labels PDF file for multiple orders at once
* Add Auto-Print script to generated PDFs
* Link previously created shipments to orders
* Easy access to tracking information for admins and customers
* Automatically notify customers about selected tracking events
* Automatically complete order on selected tracking events or when shipping label has been printed
* Test your setup before going into production by running in the sandbox

== Installation ==
1. Go to Wordpress -> Plugins -> Add New
2. Hit **Upload Plugin** button
3. Hit **Choose File** button and select zip file with the plugin
4. Hit **Install Now** button
5. Hit **Activate** link
6. Installation complete
